How to Verify EXIF Data

On macOS:

  • Open the downloaded file in Preview
  • Go to Tools → Show Inspector (⌘I)
  • Click the 'i' tab to see EXIF metadata
  • Or use Terminal: mdls filename.jpg | grep -i date
  • Or use Terminal: mdls filename.jpg | grep -i gps

On Windows:

  • Right-click the file → Properties → Details
  • Look for Date Taken and GPS coordinates

Using ExifTool (recommended):

  • Install: brew install exiftool (macOS) or download from exiftool.org
  • Run: exiftool filename.jpg
  • Check for: DateTimeOriginal, CreateDate, GPS coordinates
